# Art Image Classifier

A deep learning project that classifies an art image into one of five categories: **drawings, engraving, iconography, painting, sculpture**. Built with TensorFlow / Keras, served through a Flask web app with a custom UI.

> Final project for *Advanced Machine Learning, Deep Learning* at **Parami University**.
> Author: **May Thaw Tar**.

---

## Overview

Two convolutional neural networks are trained from scratch on the [Art Images dataset](https://www.kaggle.com/datasets/thedownhill/art-images-drawings-painting-sculpture-engraving) (8,685 images across five classes). The best-performing model is saved and served through a Flask app where users can upload an image and see the predicted category, confidence score, and probabilities for all five classes.

### Key features
- **Multi-class image classification** (5 classes)
- **Two CNN architectures compared**: a baseline simple CNN and a deeper CNN with BatchNormalization and Dropout
- **Strong data augmentation** (rotation, shift, shear, zoom, brightness, horizontal flip)
- **EarlyStopping + ReduceLROnPlateau** callbacks for clean training
- **Flask web app** with three pages: Predict, Explore Data, Model Performance
- **Low-confidence warning**: flags predictions below 55% so out-of-distribution inputs (selfies, logos) are handled gracefully
- **Custom UI design** inspired by Monet's pond and medieval illumination

### 3. Train the models

Open `deeplearning-project-maythawtar.ipynb` in Jupyter or VS Code and run all cells top to bottom. This trains both CNNs, picks the better one by weighted F1 score, and saves:

- `model/classifier.keras` (the trained best model)
- `model/metrics.pkl` (class names, image size, per-model metrics)
- `static/plots/*.png` (charts used in the web app)

Training time on CPU is roughly 30 to 50 minutes. On GPU, expect 5 to 10 minutes.

### Training pipeline
- Augmentation: rotation 25°, width and height shift 10%, shear 15%, zoom 25%, brightness jitter, horizontal flip
- Validation uses rescaling only, no augmentation, for a fair held-out measurement
- Callbacks: `EarlyStopping(patience=6, restore_best_weights=True)` and `ReduceLROnPlateau(factor=0.5, patience=3)`

### Evaluation
Models are compared on **accuracy, weighted F1, weighted precision, weighted recall** on the validation set. The best model by F1 is saved as `model/classifier.keras` and loaded by the Flask app.
