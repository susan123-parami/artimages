Image Classification

Classify an artwork into one of five categories.

A convolutional neural network trained on 8,685 art images. Upload an image and the model returns a category along with the full probability distribution.

01

Choose an image

PNG or JPG of a drawing, engraving, icon, painting, or sculpture.

02

Run the model

The image is resized to 128×128 and passed through a trained CNN.

03

Read the result

A predicted class, a confidence score, and probabilities for all five categories.

Note. The model only recognises five categories: drawings, engraving, iconography, painting, sculpture. Inputs outside this set (a photograph of a person, a logo, a symbol) will still receive a prediction, but the result page will flag it as low confidence.
